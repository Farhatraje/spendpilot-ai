# SpendPilot AI

SpendPilot AI is a multi-tool AI subscription audit platform that helps startups and teams optimize AI software spending.

Users can:
- Audit multiple AI tools
- Discover cost-saving opportunities
- Generate AI-powered summaries
- Share public audit reports
- Save reports and leads

---

# Features

- Multi-tool AI stack auditing
- AI-generated optimization summaries
- Public shareable report URLs
- Supabase backend integration
- Dynamic report pages
- Lead capture system
- Persistent local storage
- Modern SaaS UI

---

# Tech Stack

- Next.js 15
- React
- TypeScript
- Tailwind CSS
- Supabase
- OpenAI API
